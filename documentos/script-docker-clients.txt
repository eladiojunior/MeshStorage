Para mostrar que a solução é bem flexivel, aqui vamos criar uma imagem simples
e publicar vários clients em Docker... com volumes de disco diferentes.

1. Criar imagem do "meshstorage-client" e enviar para o Docker;
Acesse a pasta do projeto "meshstorage" e execute:
/> docker build -f docker/meshstorage-client.Dockerfile -t meshstorage-client:1.0.0 .

2. No Docker execute o(s) clients:
/> docker run -d --name meshstorage-client-x --restart unless-stopped \
-e SERVER_NAME=LINUX_DOCKER_X -e STORAGE_NAME=STORAGE_X \
-e URL_WEBSOCKET_SERVER=ws://192.168.1.21:3001/server-storage-websocket \
-v /mnt/WinDriveD/storage_X:/storage meshstorage-client:1.0.0

3. Criei um Docker-Compose para subir 5 clients:
/> docker-compose up -d
